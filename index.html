⚙️ Setup Instructions

  1. Create a GitHub account at github.com if you don't have one.
  2. Create a new public repository called sniper-control.
  3. Upload sniper_config.json to the repo root (the file came with this panel).
  4. Also upload index.html (this file) to the same repo.
  5. Go to Settings → Pages, set source to main branch / root. GitHub will give you a URL like https://yourname.github.io/sniper-control — bookmark this on your phone.
  6. Get the raw URL of your JSON: open sniper_config.json in GitHub, click Raw, copy the URL. It looks like:
    https://raw.githubusercontent.com/yourname/sniper-control/main/sniper_config.json
  7. Paste that URL in the box at the top of this panel and tap Load.
  8. In MetaTrader, add AISniperV11_RemoteControl.mqh to your EA (instructions in that file). Set the RemoteConfigURL input to the same raw URL.
  9. When you want to pause the EA, flip the master toggle and tap Save & Push. The EA polls every 5 minutes and will stop trading.

How saving works: This panel generates the updated JSON and opens GitHub's web editor pre-filled with the new config. Paste it in and commit — no coding needed.
