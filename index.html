⚙️ Setup Instructions

  1. Create a GitHub account at github.com if you don't have one.
  2. Create a new public repository called sniper-control.
  3. Upload sniper_config.json to the repo root.
  4. Also upload index.html (this file) to the same repo.
  5. Go to Settings → Pages, set source to main branch / root.
  6. Get the raw URL of your JSON: open sniper_config.json in GitHub, click Raw, copy the URL:
    https://raw.githubusercontent.com/yourname/sniper-control/main/sniper_config.json
  7. Paste that URL in the 📁 box at the top and tap Load.
  8. Create a GitHub Personal Access Token (PAT):
    Go to github.com → Settings → Developer settings → Personal access tokens → Tokens (classic).
    Click Generate new token, give it a name, tick repo scope, and generate. Copy the token (starts with ghp_) and paste it in the 🔑 Token box.
  9. In MetaTrader, set the EA's RemoteConfigURL input to the same raw URL. The EA polls every 5 minutes.
  10. To pause the EA: flip the master toggle and tap Save & Push. The config is pushed directly to GitHub — no redirects, no copy-paste.

⚠️ Security note: Your token is stored only in your browser's localStorage and never sent anywhere except the GitHub API.
